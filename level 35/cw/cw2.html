CSS წერის სტილი: Inline

ეს ტექსტი აიღო Inline CSS-ის სტილით, რომელიც სტილს პირდაპირ აღნიშნული HTML ელემენტში, ანუ ტეგში, ინლაინის სახით აღწერილია. Inline CSS საშუალებას გვაძლევს დაუყოვნებლივ მივცეთ სტილი კონკრეტულ ელემენტს.

CSS წერის სტილი: External

ეს ტექსტი აიღო External CSS-ის სტილით, რომელიც ჩვეულებრივ ფაილშია განთავსებული და იმპორტირდება HTML ფაილში `` ტეგის საშუალებით. External CSS გამოიყენება, როცა ბევრ HTML ფაილზე ვრცელდება ერთი და იგივე სტილი, რაც ეფექტურია შინაარსის განაწილებისთვის.
