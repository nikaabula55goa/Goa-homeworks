ag ვებ განვითარების კონტექსტში არის HTML ელემენტი, რომელიც გამოიყენება ვებგვერდის სტრუქტურის დასაგეგმად და მისი კონტენტის განსასაზღვრად. Tags გამოიყენება დოკუმენტის ტექსტის, სურათების, ბმულების, ფორმატირების და სხვა ელემენტების ორგანიზებისთვის. HTML tag-ები მოიცავს ღია და დახურულ ტეგებს: tag არის ორი სახის ერთი არის გამხსნელი მეორე დამხურავი გამხსნელი "<" იწყება ამით და მთავრდება ">"ამით სადაც მითითებულია button-ღილაკის სახელი დახურული tag თითქმის იგივეა, მაგრამ “/” გამოიყენება tag-ის დასასრულებლად.
